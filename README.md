# -Google-Translator-Bro-Python-GUI-App-
A modern and simple desktop translation application built using Python and Tkinter. This app allows users to translate text between multiple languages with a clean UI and Dark/Light mode support.


🚀 Features: 

1-🌍 Translate text between 100+ languages

2-🔄 Automatic language detection (source)

3-🖥️ User-friendly GUI (Tkinter)

4-🌙 Toggle between Dark Mode & Light Mode

5-📜 Scrollable text areas for input/output

6-⚡ Fast translation using deep_translator

7-🎯 Error handling with popup messages

🛠️ Technologies Used:

1- Python 🐍

2- Tkinter (GUI)

3- googletrans (language list)

4- deep_translator (translation engine)

📦 Installation:

Install required libraries:

1- pip install googletrans==4.0.0-rc1

2- pip install deep-translator

⚠️ Notes:

1-Make sure image files (images.png, download.png) are in the same directory.

2-Internet connection is required for translation.

3- f a language is not supported, an error message will appear.
